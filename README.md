## Features

- ⚡ **Blazing Fast** - Optimized images (WebP), fonts, and code splitting
- 🎨 **Catppuccin Theme** - Beautiful dark/light mode support
- 📝 **MDX Content** - Blog posts, projects, and philosophy writings
- 🔍 **SEO Optimized** - Structured data, meta tags, and sitemap
- 📱 **Mobile First** - Smooth touch interactions and responsive design
- 🚀 **Performance** - Lighthouse scores 95-100

## Tech Stack

- [SvelteKit](https://kit.svelte.dev/) - Full-stack Svelte framework
- [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS
- [MDSvex](https://mdsvex.pngwn.io/) - Markdown in Svelte
- [Shiki](https://shiki.matsu.io/) - Code syntax highlighting
- [Vercel](https://vercel.com/) - Deployment

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Project Structure

```
src/
├── routes/          # SvelteKit pages
├── components/      # Reusable components
├── lib/            # Utilities and config
└── content/        # MDX content (projects, blog)
```
